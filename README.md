# ProctorLock

**ProctorLock** is a full-stack secure online examination platform designed to provide a robust and reliable testing environment. It empowers educational administrators with powerful management tools, supports examiners with real-time monitoring capabilities, and ensures a secure and fair experience for candidates.

The platform is built with a modern technology stack, including **Next.js**, **React**, **TypeScript**, **Tailwind CSS**, and **Shadcn UI**, ensuring a highly modular, scalable, and performant application.
