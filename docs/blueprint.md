# **App Name**: ProctorLock

## Core Features:

- Secure Authentication: JWT-based authentication to verify users and protect access to exam sessions.
- Role-Based Access Control: Admin, examiner, and candidate roles with specific permissions and restrictions.
- Enforced Fullscreen Mode: Mandatory fullscreen mode during exams to prevent unauthorized access.
- Focus Detection: Track tab and window focus to ensure the candidate remains focused on the exam.
- Activity Monitoring: Log keyboard and mouse activity events with timestamps.
- Webcam and Mic Check: Verify webcam and microphone permissions before the exam starts.
- Periodic Snapshot Capture: Capture snapshots from the webcam at regular intervals during the exam.

## Style Guidelines:

- Primary color: Black (#000000) to convey trust and security.
- Background color: White (#FFFFFF) to provide a clean and distraction-free interface.
- Accent color: Gray (#808080) for important actions and highlights to give an impression of calm assurance.
- Body and headline font: 'Inter', a sans-serif font, to ensure readability and a modern appearance.
- Simple, minimalist icons from 'Lucide' icon set to maintain clarity and focus.
- Clean, centered layout with minimal distractions, optimized for exam focus.
- No animations for a fast and stable user experience.
